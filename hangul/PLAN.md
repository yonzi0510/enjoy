# 가나다 한글 놀이터 — 구축 계획

## 1. 원본 앱 조사: 핑크퐁! 가나다 한글

스마트스터디(더핑크퐁컴퍼니)의 유아용 한글 학습 앱. 조사로 파악한 핵심 구성:

| 원본 기능 | 설명 |
|---|---|
| 한글 동요 애니메이션 (16종) | 가나다라~아야어여, 글자 모양·소리를 노래로 익힘 |
| 자음·모음 학습 | ㄱ~ㅎ, ㅏ~ㅣ 글자별 소리 듣기 + 대표 단어 |
| 따라쓰기 | 획순 안내를 따라 손가락으로 글자 완성, 발음 반복 듣기 |
| 학습 게임 | 점프(단어 모으기), 로켓, 거품 터뜨리기 등 복습 게임 |
| 낱말 카드 수집 | 학습을 끝낼 때마다 보상으로 낱말 카드 지급 |

## 2. 로컬 재구축 방침

기존 `play/`·`english/`와 동일한 방식: **의존성 없는 바닐라 JS PWA**,
서버·로그인·외부 리소스 없음(그림은 이모지, 소리는 Web Speech TTS ko-KR + Web Audio 효과음),
GitHub Pages로 배포, 진행도는 localStorage.

## 3. 기능 매핑 (원본 → 로컬 구현)

1. **글자 배우기** — 자음 14자 + 모음 10자 그리드 → 글자 상세(이름·소리 듣기, 이모지 낱말 3개 터치해 듣기)
2. **따라쓰기** — 캔버스에 회색 안내선 + 시작점·화살표, 획순대로 드래그하면 색이 채워짐. 완성 시 발음 재생 + 낱말 카드 보상
3. **한글 놀이(복습 게임)**
   - 🫧 거품 놀이: 떠오르는 거품 중 목표 글자를 찾아 터뜨리기 (5라운드)
   - 🔤 첫소리 놀이: 이모지 낱말을 듣고 첫 자음 고르기 (5라운드)
4. **가나다 노래** — 글자 그리드가 순서대로 반짝이며 TTS가 리듬감 있게 "기역! 가, 가방~" 낭송
5. **낱말 카드책** — 따라쓰기·게임 보상으로 모은 카드를 진열, 터치하면 다시 들려줌

## 4. 파일 구조

```
hangul/
  index.html            화면 골격 (홈/그리드/상세/따라쓰기/놀이/노래/카드책)
  manifest.webmanifest  PWA 설치용
  icon-192.png icon-512.png
  css/style.css
  js/data.js            자음·모음 데이터: 이름, 획 경로(정규화 좌표), 낱말+이모지
  js/audio.js           TTS(ko-KR) + Web Audio 효과음
  js/progress.js        localStorage 진행도(따라쓴 글자, 카드, 별)
  js/trace.js           따라쓰기 캔버스 엔진(획 리샘플·근접 판정)
  js/games.js           거품 놀이 + 첫소리 놀이
  js/song.js            가나다 노래
  js/cards.js           카드책
  js/app.js             화면 전환·홈
  tools/validate-data.js  데이터 검증(Node)
  tools/e2e.mjs           Playwright 종단 테스트
```

## 5. 테스트 계획

- `node tools/validate-data.js` — 글자 24자 전부에 이름·획·낱말이 있는지, 좌표 범위, 이모지 존재 검증
- `node tools/e2e.mjs` — Playwright(Chromium)로 실제 구동:
  홈 → 글자 그리드 → 상세 → 따라쓰기(포인터 드래그로 전 획 완성) → 카드 보상 확인
  → 거품 놀이 5라운드 → 첫소리 놀이 5라운드 → 카드책에 수집 확인
- 테스트 훅: `window.__hangulTest` 로 현재 목표 글자·획 좌표를 노출해 결정적 테스트 가능
