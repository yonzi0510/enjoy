# enjoy — 은아의 놀이터

5세 아이를 위한 놀이 앱 모음 (GitHub Pages 배포)

- `/play/` 🔍 찾기 놀이터 — 숨은그림·다른그림·글자찾기·스티커놀이 (하루 30분 제한)
- `/english/` 🗣️ 영어 놀이터 — 음성으로 물어보면 영어로 답하는 단어 앱 (958단어)
- `/pixel/` 🧩 픽셀 놀이터 — 숫자 픽셀을 드래그로 칠하는 픽셀 아트 (도안 8개, 폭탄·마법봉 부스터)
- `/hangul/` 🌟 가나다 한글 놀이터 — 자음·모음 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 (핑크퐁 가나다 한글의 핵심 기능을 로컬로 재구축)
- `/japanese/` 🌸 あいうえお 일본어 놀이터 — 히라가나 46자 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 90장 (한글 놀이터와 같은 구성)
  - 히라가나 획순 데이터는 [KanjiVG](http://kanjivg.tagaini.net) (© Ulrich Apel, [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/))를 변환해 사용
- `/practika/` 🎙️ 프랙티카 놀이터 — AI 튜터와 음성으로 대화하며 회화를 연습하는 앱. **영어·일본어·중국어** 3개 언어(언어별 시나리오 롤플레이, 실시간 발화 채점·피드백, 어휘 복습, 스트릭·XP·젬 게임화). 일·중은 문자 단위 채점 지원. [Praktika](https://praktika.ai)의 핵심 기능을 로컬로 재구축
  - 소리(음성 합성)는 브라우저 TTS를 쓰므로 카카오톡 등 인앱 브라우저에서는 안 날 수 있고, 이 경우 Chrome·Safari로 열도록 안내 배너가 뜬다
- `/color/` → 픽셀 놀이터로 자동 이동 (예전 색칠 놀이터 주소)
