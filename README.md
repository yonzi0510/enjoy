# enjoy — 은아와 서하의 놀이터

5세 아이를 위한 놀이 앱 모음 (GitHub Pages 배포).
홈에서 🐰 은아 / 🐻 서하 중 누가 놀 건지 고르면 **진행도(카드·별·작품·XP)가 아이별로 따로** 저장된다.

- `/play/` 🔍 찾기 놀이터 — 숨은그림·다른그림·글자찾기·스티커놀이
- `/english/` 🗣️ 영어 놀이터 — 음성으로 물어보면 영어로 답하는 단어 앱 (958단어)
- `/pixel/` 🧩 픽셀 놀이터 — 숫자 픽셀을 드래그로 칠하는 픽셀 아트 (도안 8개, 폭탄·마법봉 부스터)
- `/hangul/` 🌟 가나다 한글 놀이터 — 자음·모음 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 (핑크퐁 가나다 한글의 핵심 기능을 로컬로 재구축)
- `/japanese/` 🌸 あいうえお 일본어 놀이터 — 히라가나 46자 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 90장 (한글 놀이터와 같은 구성)
  - 히라가나 획순 데이터는 [KanjiVG](http://kanjivg.tagaini.net) (© Ulrich Apel, [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/))를 변환해 사용
- `/practika/` 🎙️ 프랙티카 놀이터 — AI 튜터와 음성으로 대화하며 회화를 연습하는 앱. **영어·일본어·중국어** 3개 언어(언어별 시나리오 롤플레이, 실시간 발화 채점·피드백, 어휘 복습, 스트릭·XP·젬 게임화). 일·중은 문자 단위 채점 지원. [Praktika](https://praktika.ai)의 핵심 기능을 로컬로 재구축
  - 소리(음성 합성)는 브라우저 TTS를 쓰므로 카카오톡 등 인앱 브라우저에서는 안 날 수 있고, 이 경우 Chrome·Safari로 열도록 안내 배너가 뜬다
- `/write/` ✍️ 글씨 놀이터 — 패드+펜슬로 깍두기노트(원고지 칸)에 필사하는 글씨 연습 앱. 회색 글자를 따라 쓰고 아랫줄에 혼자 써 보는 구성(느슨한 겹침 판정), 크레용 10색+무지개 펜, 닿은 획만 지우는 부분 지우개. 자음·모음, 가나다라~기니디리 여섯 글자 줄, 낱말 62개(주제별 6묶음), 전래 동요·동화(듣고 쓰기), 🎧 받아쓰기 7단계 230개(듣고 빈 칸에 쓰기 → 정답 공개 → 스스로 확인), 🎨 자유 낙서장(펜 4종·이모지 스티커 54종·보관)까지. 쓴 글씨는 작품 갤러리에 보관. 🎤 물어보고 쓰기: 마이크에 "토끼는 어떻게 써?" 하고 물으면 그 낱말의 필사 페이지가 열린다 (음성 인식이 안 되는 브라우저에서는 직접 입력)
- `/color/` → 픽셀 놀이터로 자동 이동 (예전 색칠 놀이터 주소)
- `/parent/` 🔑 부모님 페이지 — 확인 번호(기본 1234) 뒤에서 하루 사용 시간 제한(기본 30분, 모든 앱 공용)·확인 번호 변경·앱 노출(프랙티카/일본어/픽셀 활동지/받아쓰기 6~7단계)·음성 인식(마이크) 허용을 설정하고, 진행도를 JSON 파일로 내려받거나 되살릴 수 있다

## 5세 안전 기본값

- **하루 사용 시간 제한이 모든 앱 공용**이다(`shared/time-limit.js`). 다 쓰면 잠기고 부모 확인 번호로 30분씩 연장.
- 홈 카드를 누르면 **앱 이름을 소리로 읽어 준 뒤** 이동한다 (글을 못 읽는 아이 배려).
- **프랙티카는 기본 숨김**(성인 회화 내용), **일본어는 잠금**(한글 낱말 카드 10장을 모으면 자동 해제), **픽셀 활동지·받아쓰기 6~7단계는 기본 숨김** — 모두 부모님 페이지에서 열 수 있다.
- 🗣️ 목소리 설정은 **길게 눌러야** 열린다 (아이 오조작 방지).
- 서비스 워커(`sw.js`)로 한 번 연 놀이터는 **오프라인에서도 열린다**.
- **아이 프로필**: 홈 상단에서 은아/서하를 고른다. 각 앱 구석의 작은 배지가 지금 누가 노는지 보여준다.
  하루 시간 제한은 기기 공용(둘이 함께 소진 — 프로필을 바꿔도 초기화되지 않음)이다.
- 🐣 **학습 펫** (`shared/pet.js`) — 학습을 완료하면 간식(페이지·카드·게임 한 판)과 식사(챕터·도안 완성)가 생기고, 먹여 주면 펫이 자란다: 🥚 알 → 아기 → 어린이 → 어른. 다 크면 📖 도감에 등록되고 새 알이 도착 — **16종**(강아지·고양이·판다·유니콘 등)을 수집한다. 부화하면 **이름을 지어 주고**(추천 칩 + 직접 입력, ✏️로 언제든 변경), 먹이 8번마다 🎁 **꾸미기 선물**(왕관·리본·풍선 등 12종)을 받아 입힌다. 글씨·픽셀·한글·일본어 놀이터와 메인 홈에서 만날 수 있고, 은아·서하가 각자 키운다 (나쁜 상태 없음 — 자라기만 한다)
