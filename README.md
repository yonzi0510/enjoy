# enjoy — 은아와 서하의 놀이터

5세 아이를 위한 놀이 앱 모음 (GitHub Pages 배포).
홈에서 🐰 은아 / 🐻 서하 중 누가 놀 건지 고르면 **진행도(카드·별·작품·XP)가 아이별로 따로** 저장된다.

- `/play/` 🔍 찾기 놀이터 — 숨은그림·다른그림·글자찾기·스티커놀이
- `/english/` 🗣️ 영어 놀이터 — 음성으로 물어보면 영어로 답하는 단어 앱 (958단어)
- `/pixel/` 🧩 픽셀 놀이터 — 숫자 픽셀을 드래그로 칠하는 픽셀 아트 (도안 8개, 폭탄·마법봉 부스터)
- `/hangul/` 🌟 가나다 한글 놀이터 — 자음·모음 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 (핑크퐁 가나다 한글의 핵심 기능을 로컬로 재구축)
- `/japanese/` 🌸 あいうえお 일본어 놀이터 — 히라가나 46자 배우기, 획순 따라쓰기, 거품·첫소리 게임, 낱말 카드 90장 (한글 놀이터와 같은 구성)
  - 히라가나 획순 데이터는 [KanjiVG](http://kanjivg.tagaini.net) (© Ulrich Apel, [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/))를 변환해 사용
- `/practika/` 🎙️ 프랙티카 놀이터 — AI 튜터와 음성으로 대화하며 회화를 연습하는 앱. **영어·일본어·중국어** 3개 언어(언어별 시나리오 롤플레이, 실시간 발화 채점·피드백, 어휘 복습, 스트릭·XP·젬 게임화). 일·중은 문자 단위 채점 지원. [Praktika](https://praktika.ai)의 핵심 기능을 로컬로 재구축
  - 소리(음성 합성)는 브라우저 TTS를 쓰므로 카카오톡 등 인앱 브라우저에서는 안 날 수 있고, 이 경우 Chrome·Safari로 열도록 안내 배너가 뜬다
- `/write/` ✍️ 글씨 놀이터 — 패드+펜슬로 깍두기노트(원고지 칸)에 필사하는 글씨 연습 앱. 회색 글자를 따라 쓰고 아랫줄에 혼자 써 보는 구성(느슨한 겹침 판정), 크레용 10색+무지개 펜, 닿은 획만 지우는 부분 지우개. 자음·모음, 가나다라~기니디리 여섯 글자 줄, 낱말 62개(주제별 6묶음), 전래 동요·동화(듣고 쓰기), 🎧 받아쓰기 7단계 230개(듣고 빈 칸에 쓰기 → 정답 공개 → 스스로 확인), 🎨 자유 낙서장(펜 4종·이모지 스티커 54종·보관)까지. 쓴 글씨는 작품 갤러리에 보관. 🎤 물어보고 쓰기: 마이크에 "토끼는 어떻게 써?" 하고 물으면 그 낱말의 필사 페이지가 열린다 (음성 인식이 안 되는 브라우저에서는 직접 입력)
- `/math/` 🔢 산수 놀이터 — 5세 산수 연습. ✏️ 숫자 따라쓰기 1~100(펜슬로 깍두기 칸에 쓰고, 십 묶음 🧺 + 낱개 그림으로 수 세기), 🐸 숫자 징검다리(개구리가 돌 위를 폴짝 이동하며 더하기·빼기를 '이동'으로 익히는 개념 단계 — 도착할 돌을 누르면 한 칸씩 뛰어 보여주고 식이 짠! 공개), 🍎 그림 덧셈·🍏 그림 뺄셈(풀 때는 식을 숨기고 그림만 크게 → 정답 후 식이 나타나 연결), ➕➖ 숫자 문제 3단계(5까지/10까지/20까지, 한 판 5문제·보기 3개, 틀리면 그림 힌트가 저절로). 정답 후에는 그림을 하나씩 같이 세는 이해 단계를 거쳐 아이가 직접 ▶를 눌러야 다음 문제로. 오답 벌점 없음, 학습 펫 먹이 연동
- `/shape/` 🔷 도형 놀이터 — 손가락으로 조각을 끌어다 맞추는 5세 도형 퍼즐 세 가지. 🧩 칠교놀이 도안 10개(집·배·물고기 등 — 쉬운 단계 5개는 조각이 맞는 방향으로 나와 끌어다 놓기만, 빙글빙글 단계 5개는 조각을 톡 누르면 45°씩 회전), 🐢 블록 퍼즐 6개(거북이·나비·로켓 등 — 여러 칸짜리 블록을 격자에 스냅해 색 그림 완성), 🏠 도형 맞추기 6개(집·기차·성 등 — 세모·네모·동그라미·반원·사다리꼴을 같은 모양 자리에, 틀린 자리는 부드럽게 튕겨 돌아옴). 제 자리 근처(판의 5% 안팎)에 놓으면 착! 붙고, 완성하면 색종이 축하 + 별 + 학습 펫 먹이(빙글빙글 칠교는 식사)
- `/market/` 🛒 시장 놀이터 — 가게 놀이로 돈 개념을 익히는 앱. 손님(펫 도감 친구들)이 걸어와 주문하면 상품을 장바구니에 담고, 가격표(동전 그림)만큼 100원·500원 동전을 계산대에 드래그해 지불. 3단계(100원만 → 500+100 조합 → 두 상품 합산), 초과·부족은 부드럽게 안내, 한 판 손님 5명 접객
- `/lab/` 🧪 색깔 실험실 — 물감 방울을 유리병에 드래그해 섞는 혼색 과학 놀이. 미션 12색(주황·초록·보라 등) 도감 수집 + 자유 실험 선반(만든 색에 "노을 주황" 같은 이름표), RYB 혼색 엔진으로 빨+노=주황이 자연스럽게, 오답이면 "조금 더 노랑을 넣어 볼까?" 힌트
- `/bag/` 💡 생각 놀이터 — 본보기를 보고 조각을 끌어다 똑같이 맞추는 관찰·소근육 놀이 모음(URL은 `/bag/`). 🥄 요리조리 숟가락(끌어다 놓고 톡 눌러 돌려 방향·위치 맞추기, 30개) · 🎚️ 요리조리 빨대(색 슬라이더를 밀어 본보기 높이에 맞추기 + 실로폰 음, 30개) · 🟧 네모 조각 맞추기(조각 돌려 무늬 맞추기, 30개) · 🎈 요리조리 풍선 줄(본보기 줄 모양을 펜슬로 따라 그리기 3단계 30개 — 재현율·정밀도로 판정해 아무렇게나 그리면 통과 안 됨)
- `/coloring/` 🎨 색칠공부 — 밑그림이 **흐린 점선 안내선**으로 나와 아이가 크레용으로 **따라 그린 뒤 색칠**한다. 🪣 물통 채우기(영역을 톡 누르면 색이 차오름, 캔버스 flood fill — 보이지 않는 실선 벽으로 막아 점선 틈으로 색이 새지 않음) · 🖍️ 크레용(자유 브러시), 12색+무지개·되돌리기·전체지우기. 밑그림 30장(동물·과일음식·탈것사물·자연계절), 완성작은 갤러리에 보관
- `/burger/` 🍔 햄버거 가게 — 미션 카드에 나온 순서(아래빵→속재료→윗빵)대로 재료를 꼬치에 쌓아 햄버거를 완성하는 순서·기억 놀이. 재료 12종·미션 30개(단계별 3·5·7층), 틀린 재료는 부드럽게 튕김(무벌점)
- `/color/` → 픽셀 놀이터로 자동 이동 (예전 색칠 놀이터 주소)
- `/parent/` 🔑 부모님 페이지 — 확인 번호(기본 1234) 뒤에서 하루 사용 시간 제한(기본 30분, 모든 앱 공용)·**오늘 사용 시간 지금 초기화**(🔄, 잠겨 있어도 다시 놀 수 있게)·확인 번호 변경·앱 노출(프랙티카/일본어/픽셀 활동지/받아쓰기 6~7단계)·음성 인식(마이크) 허용을 설정하고, 진행도를 JSON 파일로 내려받거나 되살릴 수 있다. (사용 시간은 매일 **로컬 자정**에 자동 초기화 — 예전엔 UTC 기준이라 KST 오전 9시에 넘어가던 버그를 고침)

## 5세 안전 기본값

- **하루 사용 시간 제한이 모든 앱 공용**이다(`shared/time-limit.js`). 다 쓰면 잠기고 부모 확인 번호로 30분씩 연장.
- 홈 카드를 누르면 **앱 이름을 소리로 읽어 준 뒤** 이동한다 (글을 못 읽는 아이 배려).
- **프랙티카는 기본 숨김**(성인 회화 내용), **일본어는 잠금**(한글 낱말 카드 10장을 모으면 자동 해제), **픽셀 활동지·받아쓰기 6~7단계는 기본 숨김** — 모두 부모님 페이지에서 열 수 있다.
- 🗣️ 목소리 설정은 **길게 눌러야** 열린다 (아이 오조작 방지).
- 서비스 워커(`sw.js`)로 한 번 연 놀이터는 **오프라인에서도 열린다**.
- **아이 프로필**: 홈 상단에서 은아/서하를 고른다. 각 앱 구석의 작은 배지가 지금 누가 노는지 보여준다.
  하루 시간 제한은 기기 공용(둘이 함께 소진 — 프로필을 바꿔도 초기화되지 않음)이다.
- 🐣 **학습 펫** (`shared/pet.js` + 캐릭터 `shared/pet-avatar.js`) — 펫은 이모지가 아니라 **직접 그린 SVG 캐릭터**다: 공통 리그 위에 종별 귀·꼬리·특징을 얹은 16종, 숨쉬기·눈 깜빡임·귀 쫑긋·꼬리 살랑·폴짝 잔동작이 살아 있고, 먹이를 주면 입을 벌려 오물오물 먹고 하트를 날리며, 만지면 간지럼을 탄다. 알은 클수록 금이 가고 흔들린다 (검수용 데모: `shared/pet-avatar-demo.html`). 학습을 완료하면 간식(페이지·카드·게임 한 판)과 식사(챕터·도안 완성)가 생기고, 먹여 주면 펫이 자란다: 🥚 알 → 아기 → 어린이 → 어른. 다 크면 📖 도감에 등록되고 새 알이 도착 — **16종**(강아지·고양이·판다·유니콘 등)을 수집한다. 부화하면 **이름을 지어 주고**(추천 칩 + 직접 입력, ✏️로 언제든 변경), 먹이 8번마다 🎁 **꾸미기 선물**(왕관·리본·풍선 등 12종)을 받아 입힌다. 글씨·픽셀·한글·일본어 놀이터와 메인 홈에서 만날 수 있고, 은아·서하가 각자 키운다 (나쁜 상태 없음 — 자라기만 한다)
