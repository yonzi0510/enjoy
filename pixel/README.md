# 픽셀 놀이터 — Pixel Art 스타일 픽셀 컬러 바이 넘버 (로컬 PWA)

## 1. Pixel Art(Easybrain) 조사 요약

"Pixel Art - Color by Number"는 픽셀 격자 컬러 바이 넘버 앱이다. 핵심 작동 방식:

1. **도안 갤러리** — 완성 모습(픽셀 미리보기) 썸네일과 진행률이 보이는 라이브러리.
2. **숫자 격자** — 그림이 W×H 픽셀 격자로 되어 있고 각 칸에 팔레트 번호가 표시됨.
   두 손가락으로 확대해야 숫자가 보일 만큼 칸이 커진다 (축소하면 숫자가 사라지고 색만 보임).
3. **탭 + 드래그 채색** — 색을 고른 뒤 칸을 탭하거나 **한 손가락으로 쓸면 지나간 칸이 연속으로**
   칠해진다. 두 손가락은 핀치 줌/이동.
4. **오답은 물 빠진 색** — 틀린 번호 칸을 칠하면 그 색의 washed(연한) 색으로 표시되고
   숫자가 남아 있어 올바른 색으로 다시 칠해 고칠 수 있다.
5. **색상별 진행률** — 팔레트에 진행 표시, 한 색을 다 칠하면 ✓ 후 다음 색 자동 선택.
6. **부스터** — 💣 폭탄(탭 주변 영역을 올바른 색으로 일괄 채색), 🪄 마법봉(한 번호의 남은 칸 전부).
7. **자동 저장·이어하기**, **완성 연출**(격자·숫자가 사라지며 그림 공개) + 내 작품 갤러리.

## 2. 로컬 앱 재구축 계획

`color/`(색칠 놀이터)와 동일하게 **서버·로그인·외부 라이브러리 없는 정적 PWA**.
격자 렌더링은 SVG 대신 **<canvas>** (드래그 채색·수백 칸에 유리). 데이터는 localStorage.

### 구조

```
pixel/
  index.html            # 홈(갤러리) · 채색 화면 · 완성 오버레이 · 내 작품
  manifest.webmanifest  # PWA 설치
  icon-192.png / icon-512.png   # 픽셀 토끼 아이콘
  css/style.css
  js/
    progress.js         # localStorage (칸 상태·부스터·완성 기록)
    audio.js            # WebAudio 효과음 + 한국어 음성
    engine.js           # canvas 렌더러·줌/팬·드래그 채색·부스터·완성 연출
    pictures/*.js       # 도안 (그림 1개 = 파일 1개)
  tools/
    validate-picture.js # 도안 계약 정적 검증기 (node)
    e2e.js              # Playwright 처음~끝 자동 플레이 검증
```

### 도안 계약 (pictures/*.js)

```js
PIXELS.push({
  id: 'heart', name: '반짝 하트', emoji: '❤️', category: 'shape',
  palette: ['#FFF3E0', '#FF4D6D', ...],   // 번호 = 인덱스+1 (0번도 칠하는 배경색)
  rows: [ '000000000000', '001100001100', ... ],  // 문자 '0'~'9' = palette 인덱스
});
```

- 모든 칸이 채색 대상(배경 포함). 격자 8×8 ~ 48×48, 색 최대 10개.
- 도안은 JS 파일이므로 원·호 같은 도형은 코드로 생성해도 된다.

### 핵심 기능 구현

| Pixel Art 기능 | 구현 |
|---|---|
| 격자+숫자 렌더 | canvas. 칸 크기(px)가 임계값 이상일 때만 숫자 표시, 축소 시 색만 |
| 선택 색 강조 | 선택 번호 미채색 칸은 진한 회색, 나머지는 연회색 |
| 드래그 채색 | 한 손가락/마우스 드래그로 지나간 칸 연속 채색 (칸 사이 보간) |
| 오답 washed | 틀린 색은 55% 흰색 혼합으로 표시, 숫자 유지 → 올바른 색으로 덮어쓰기 |
| 줌/팬 | 두 손가락 핀치+이동, 휠 줌(커서 기준), ⛶ 전체 보기 |
| 부스터 | 💣 ×3: 다음 탭 주변 5×5 자동 채색 · 🪄 ×1: 선택 번호 남은 칸 전부 |
| 색상별 진행 | 팔레트 진행 링 + ✓ + 자동 다음 색 |
| 자동 저장 | 칸 배열·부스터 잔량을 localStorage(`pixel-playground-v1`)에 저장 |
| 완성 연출 | 격자·숫자 페이드 → 전체 보기 + 폭죽·팡파레, 내 작품 갤러리, 다시 색칠 |

### 도안 라인업 (14개 · 난이도 3단계)

난이도가 오를수록 격자·색이 늘고 부스터 기본 지급도 늘어난다
(쉬움 💣3·🪄1 / 보통 💣4·🪄2 / 어려움 💣6·🪄2).

| 단계 | 도안 |
|---|---|
| 🌱 쉬움 (12~22) | ❤️ 하트 12² · ⭐ 별 14² · 🐞 무당벌레 14² · 🐰 토끼 16² · 🌷 튤립 16² · 🌈 무지개 18² · 🍦 아이스크림 20² · 🚀 로켓 22² |
| 🌟 보통 (26) | 🦋 나비 · 🍓 딸기 · 🐠 바닷속 물고기 |
| 🔥 어려움 (32) | 🐼 대나무 팬더 · 🦕 아기 공룡 · 🎂 생일 케이크 |

### 테스트 계획 (기본 기능이 처음부터 끝까지)

1. **정적 검증** — `node pixel/tools/validate-picture.js`: 격자 크기·행 길이 일치·
   문자 범위·팔레트 hex·모든 색 사용·id 중복.
2. **E2E 자동 플레이** — `node pixel/tools/e2e.js` (Playwright + 내장 Chromium):
   - 갤러리 렌더·카테고리 필터 → 도안 열기
   - 오답 탭 → washed 표시·오답 수 1 → 올바른 색으로 고치기
   - **실제 클릭으로 하트 전체(144칸) 채색** → 색상별 ✓·자동 선택 → 완성 오버레이
   - 드래그 채색: 마우스 드래그로 여러 칸이 한 번에 칠해지는지
   - 부스터: 💣 5×5 일괄 채색·잔량 감소, 🪄 한 색 전체 채색
   - 새로고침 → 진행률·부스터 잔량 복원 (이어하기)
   - 휠 줌 후에도 좌표 변환이 정확한지, 완성작 감상·다시 색칠, 콘솔 오류 0
